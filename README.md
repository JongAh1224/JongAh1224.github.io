# 나의 블로그 프로젝트

1. **github** 원격 저장소 생성 JongAh1224.github.io
2. 내 **로컬 저장소**에 클론
3. 윈도우에 **jekyll** 설치
4. 로컬 저장소에 **jekyll** 다운로드
5. **원격 저장소**에 push
6. 특강 교수님께서 주신 **테마** 사이트에서 **테마** 찾기
7. 테마 깃허브에서 **클론**
8. 클론한 테마 파일 로컬 저장소에 **덮어쓰기**
9. 테마 잘 **적용**되었는지 확인
10. **Disqus**를 활용한 댓글 기능 추가
11. **포스트** 작성
12. **포스트** 수정
13. **블로그 완성**
